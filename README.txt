Exemplo de implementação da estrutura de dados Pilha em C.

O código está organizado da seguinte forma:
   pilha.h 				- arquivo de cabeçalho com a definição das funções inerente à estrutura de dados pilha.
   pilha.c 				- implementação das funções definidas no arquivo de cabeçalho pilha.h
   ExemploUsoPilha.c 	- exemplo de uso da estrutura de dados do tipo C. 
   
   O programa de exemplo define uma pilha de 20 elementos inteiro e solicita ao usuário que informe os valores.

Para compilar
  gcc pilha.c -o pilha
  gcc ExemploUsoPilha.c -o ExemploUsoPilha
  
Para executar
   ./ExemploUsoPilha
